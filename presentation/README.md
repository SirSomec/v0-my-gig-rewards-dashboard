# Презентация для совета директоров

Презентация MyGig Rewards Dashboard в виде одностраничного HTML в фирменном стиле MyGig.

## Файлы

- **`board-presentation.html`** — презентация: откройте в браузере. Переключение слайдов: кнопки внизу справа или клавиши → ←, Space, PageDown/PageUp.
- **`speaker-notes.md`** — подсказки для докладчика: что говорить, демо, ответы на возражения.

## Как показывать

Откройте **`board-presentation.html`** в браузере (двойной клик по файлу или «Открыть файл»). Для полноэкранного режима нажмите F11.

## Структура презентации

1. Титул и повестка  
2. Проблема → Решение → Результат  
3. Бизнес-цели и выгоды  
4. Экономическая выгода и пример расчёта  
5. Демонстрация продукта  
6. Архитектура и интеграция  
7. Этапы внедрения и что ещё предстоит  
8. Риски и митигация  
9. Рекомендация и запрос решения  

Демонстрация продукта проводится **вживую** между слайдами «Демонстрация продукта» и «Архитектура».
