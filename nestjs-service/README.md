# Шаблон для микросервиса на Nestjs

## Описание
Шаблон предназначен для быстрого старта разработки и деплоя микросервисов на Nestjs.  
Содержит готовую структуру проекта, базовые скрипты и примеры.

---

## Структура
- `init.sh` — инициализация проекта
- `src/` — исходный код сервиса
- `package.json` и `tsconfig.json` — конфигурация проекта (Node.js + TypeScript)
- `dist/` — сборка проекта (создаётся автоматически)

---

## Инициализация проекта (ОБЯЗАТЕЛЬНО)
Для инициализации используйте:

```bash
npm i
```

```bash
./init.sh --name <Имя сервиса>
```

Скрипт:
- проходит по всем файлам шаблона;
- заменяет плейсхолдер `<SERVICE_NAME>` на указанное имя;
- подготавливает проект к дальнейшей разработке.

Пример:

```bash
./init.sh --name MyService
```

Так есть возможность добавить описание, см `./init.sh -h`
